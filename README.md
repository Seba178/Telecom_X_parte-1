Análisis TelecomX

Descripción
Este proyecto analiza los factores que influyen en la evasión de clientes (churn) para una empresa de telecomunicaciones, utilizando Python y análisis exploratorio de datos.

Cómo usar
pip install pandas numpy matplotlib seaborn jupyter

Ejecución
1. Clona este repositorio
2. Abre `TelecomX_LATAM.ipynb` en Jupyter Notebook o Google Colab
3. Asegúrate que `TelecomX_Data.json` esté en la misma carpeta
4. Ejecuta todas las celdas

Archivos
- `TelecomX_LATAM.ipynb` - Notebook principal con todo el análisis
- `TelecomX_Data.json` - Datos de clientes
- `TelecomX_diccionario.md` - Diccionario de datos

Principales hallazgos
- **Tasa de churn:** 26.5%
- **Clientes más propensos a irse:** Contrato mensual, pago con cheque electrónico, servicio de fibra óptica
- **Clientes más leales:** Contratos largos, pagos automáticos, con soporte técnico

Visualizaciones
El notebook incluye gráficos de:
- Distribución de churn
- Análisis por tipo de contrato
- Análisis por método de pago
- Relación con antigüedad y cargos mensuales
